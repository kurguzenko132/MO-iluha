# Для Вики

Чистая версия проекта с нейтральным названием для превью ссылки.

## Важно

В превью ссылки больше не должно быть текста “100 комплиментов”.

Метаданные:

- title: `Для Вики 💛`
- description: `Маленькая личная страница, сделанная специально для тебя.`

## Если в Telegram/VK/мессенджере всё ещё старое превью

Это кэш. Нужно:
1. Сделать новый deploy на Vercel.
2. Подождать несколько минут.
3. Отправить ссылку заново.
4. Иногда помогает добавить к ссылке `?v=2`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
